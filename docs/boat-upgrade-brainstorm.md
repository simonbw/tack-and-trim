# Boat Upgrade Brainstorm

Ideas for a boat upgrade system where players can purchase improvements for their keelboat/yacht. Organized by boat system.

## Hull & Structure

| Upgrade                | Effect                                                                          |
| ---------------------- | ------------------------------------------------------------------------------- |
| **Fiberglass Hull**    | Reduces skin friction coefficient — less drag, higher top speed                 |
| **Carbon Fiber Hull**  | Even lower friction + reduced mass                                              |
| **Wider Beam**         | More stability (less heel), but more drag                                       |
| **Narrower Beam**      | Less drag, faster, but less stable                                              |
| **Deeper Draft Hull**  | Better upwind performance, more grounding risk                                  |
| **Shallow Draft Hull** | Navigate shallower water, worse upwind                                          |
| **Reinforced Hull**    | Reduces damage/slowdown from grounding (lower hull and keel friction penalties) |
| **Longer Waterline**   | Higher theoretical hull speed                                                   |
| **Painted Bottom**     | Anti-fouling coating — prevents gradual drag increase over time                 |

## Sails

| Upgrade                           | Effect                                                                         |
| --------------------------------- | ------------------------------------------------------------------------------ |
| **Racing Mainsail**               | Higher lift scale, better acceleration and top speed                           |
| **Kevlar Sails**                  | Maintains shape better — reduced slack factor, better performance in high wind |
| **Larger Mainsail**               | More sail area (more nodes/longer boom) — more power but harder to control     |
| **Reefing System**                | Ability to reduce sail area in heavy wind to prevent overpowering              |
| **Battens**                       | Improved sail shape — better lift-to-drag ratio at all angles                  |
| **Spinnaker**                     | Third sail for downwind — massive boost when running/broad reaching            |
| **Gennaker/Asymmetric Spinnaker** | Easier-to-handle downwind sail, less power than full spinnaker                 |
| **Code Zero**                     | Ultra-light upwind reaching sail for light air conditions                      |
| **Roller Furling Jib**            | Faster hoist speed for the jib — quicker sail changes                          |
| **Whisker Pole**                  | Holds jib out when sailing downwind — better wing-on-wing performance          |
| **Upgraded Telltales**            | Better visual feedback on sail trim — wider/more visible indicators            |

## Keel & Centerboard

| Upgrade                | Effect                                                                |
| ---------------------- | --------------------------------------------------------------------- |
| **Deeper Centerboard** | Increased draft — better lateral resistance, less leeway upwind       |
| **Bulb Keel**          | Added weight at bottom — more stability, can carry more sail          |
| **Swing Keel**         | Retractable — reduce draft in shallow water, raise for speed downwind |
| **High-Aspect Keel**   | Better lift-to-drag ratio — points higher upwind                      |
| **Twin Keels**         | Can rest upright when grounded, moderate upwind performance           |

## Rudder & Steering

| Upgrade              | Effect                                                                  |
| -------------------- | ----------------------------------------------------------------------- |
| **Balanced Rudder**  | Higher max steer angle and faster steer adjust speed                    |
| **Larger Rudder**    | More turning force but slightly more drag                               |
| **Kick-Up Rudder**   | Reduces grounding damage — rudder lifts on contact                      |
| **Tiller Extension** | Allows steering from further forward — steer while managing other tasks |
| **Wheel Steering**   | Smoother, more precise steering input curve                             |

## Rigging & Lines

| Upgrade              | Effect                                                                      |
| -------------------- | --------------------------------------------------------------------------- |
| **Cam Cleats**       | Faster trim speed — quicker sheet adjustments                               |
| **Ratchet Blocks**   | Easier to hold trim in heavy wind — less sheet slip                         |
| **Dyneema Sheets**   | Lighter, lower stretch — more responsive sail trim                          |
| **Longer Mainsheet** | Greater max length — can ease sail further for deep downwind angles         |
| **Boom Vang**        | Controls boom lift — better sail shape when eased, prevents accidental jibe |
| **Cunningham**       | Controls sail draft position — tune for different wind strengths            |
| **Outhaul Upgrade**  | Flattens foot of sail — better heavy air performance                        |
| **Traveler**         | Controls boom angle independent of sheet tension — finer upwind tuning      |

## Rowing & Auxiliary Power

| Upgrade                     | Effect                                                                  |
| --------------------------- | ----------------------------------------------------------------------- |
| **Better Oars**             | Increased rowing force                                                  |
| **Sculling Oar**            | Continuous propulsion instead of discrete strokes — smoother but slower |
| **Electric Trolling Motor** | Slow but steady propulsion without effort, limited battery              |
| **Sliding Seat**            | More efficient rowing — higher force, longer duration per stroke        |

## Anchor & Mooring

| Upgrade               | Effect                                                       |
| --------------------- | ------------------------------------------------------------ |
| **Heavier Anchor**    | Holds in stronger currents, but more mass to haul            |
| **Longer Rode**       | Greater max rode length — anchor in deeper water             |
| **Windlass**          | Faster rode retrieve speed — electric anchor retrieval       |
| **Danforth Anchor**   | Better holding power in sand/mud (terrain-dependent)         |
| **Sea Anchor/Drogue** | Deployable drag device — slows drift without touching bottom |

## Navigation & Awareness

| Upgrade              | Effect                                                |
| -------------------- | ----------------------------------------------------- |
| **Wind Vane**        | Shows apparent wind direction on boat                 |
| **Anemometer**       | Displays wind speed numerically                       |
| **Compass**          | Shows heading                                         |
| **GPS/Chartplotter** | Minimap or course plotting ability                    |
| **Depth Sounder**    | Warning when approaching shallow water/grounding risk |
| **Barometer**        | Predict upcoming wind changes                         |
| **Range Finder**     | Shows distance to marks/waypoints                     |

## Performance Systems

| Upgrade                 | Effect                                                                                     |
| ----------------------- | ------------------------------------------------------------------------------------------ |
| **Hydrofoils**          | At high speed, hull lifts out of water — dramatically reduces drag above a threshold speed |
| **Canting Keel**        | Movable ballast — actively counterbalances heel for more speed                             |
| **Asymmetric Hull**     | Optimized for one tack — niche racing advantage                                            |
| **Water Ballast Tanks** | Adjustable weight distribution — trade speed for stability                                 |

## Crew & Comfort

| Upgrade            | Effect                                                                           |
| ------------------ | -------------------------------------------------------------------------------- |
| **Hiking Straps**  | Player can lean out — more righting moment in heavy wind                         |
| **Trapeze**        | Even more righting moment — enables sailing in very heavy wind without capsizing |
| **Cushioned Seat** | Pure cosmetic / humor                                                            |
| **Cup Holder**     | Pure cosmetic / humor                                                            |

## Environmental / Situational

| Upgrade         | Effect                                                 |
| --------------- | ------------------------------------------------------ |
| **Storm Jib**   | Small heavy-weather jib — maintains control in gales   |
| **Trysail**     | Small heavy-weather mainsail replacement               |
| **Weed Cutter** | Prevents fouling on keel/rudder in weedy areas         |
| **Fenders**     | Reduces damage from collisions with docks/other boats  |
| **Bilge Pump**  | If water ingress is a mechanic — pump water out faster |
